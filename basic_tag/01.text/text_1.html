제목과 텍스트 관련 태그

<h>태그는 제목을 표현하며 1~6까지의 6단계가 있다.

abbr 태그

축약어의 원래 단어를 말풍선으로 표시한다. HTML

b 태그

<b> : 굵은 글씨체로 표현

cite, mark 태그

<cite> : 책, 문구, 영화대사 등의 인용구를 표현할 때 사용한다.
<mark> : 문장에서 특정 부분을 강조할 때 하이라이트로 표현

q와 blockquote

<q> : 짧은 인용문을 인용부호로 표현할 때 사용한다.

<blockquote> :

"다른 블로그나 사이트의 글을 인용할 경우 사용한다."

strong, em 태그

<strong> : 텍스트 중에서 굵게 강조하는 표시할 때 사용
<em> : 텍스트를 기울임체로 표시하여 강조의 의미로 사용
<i> : 텍스트를 기울임체로 표시할 때 사용

time, address 태그

<time> : 시간을 의미하는 태그이다. 예, 2022-05-09
<address> : 주소를 나타내는 태그로 텍스트를 기울임체로 표시한다
서울시 강남구 역삼동 11-12

ruby, rt 태그

<ruby>, <rt> : 漢字한자

meter 태그

<meter> : 측정값을 70% 그래프로 표현하는 태그
